# Repository for the AFFIRM Trial

* data2 folder: Contains output for test_2026_02_10_Anna.qmd that is required for AFFIRM_Report_FH6.qmd.
* AFFIRM_Report_FH6.pdf: Simulation report for the trial.
* AFFIRM_Report_FH6.qmd: Contains code for generating AFFIRM_Report_FH6.pdf.
* Report.Rproj: R project for running test_2026_02_10_Anna.qmd and AFFIRM_Report_FH6.qmd.
* test_2026_02_10_Anna.qmd: Contains code for running the adaptive trial simulations.
